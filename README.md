# Ambulance Dispatch & Scheduling System

## Purpose
Web-based EMS dispatch system to:
- Log incoming transport calls
- Track patient and insurance info
- Schedule units/trucks hour-by-hour
- Assign calls with drag-and-drop interface
- Provide warnings for overlapping assignments

## Core Features
- Pending Calls Bar at top (color-coded by priority)
- Daily Schedule Grid with 15-minute increments
- Drag-and-Drop calls to units
- Call Detail Modal for full information
- Current Time Line Indicator
- Hard warning for overlapping units
- Escalation color coding as pickup time approaches

## Database Blueprint:
Tables:
1. Calls
2. Patients
3. Insurance
4. Units / Trucks
