# Dispatch & Transport Scheduling System

## Purpose
A web-based application to:
- Log incoming transport calls
- Store patient and insurance data
- Schedule units and trucks
- Assign calls to time slots

## Core Features
- Call Intake Form
- Patient Demographics
- Insurance Tracking
- Unit Scheduling Screen
- Shared Notes System

## Future Features
- Drag & Drop Scheduling
- Reporting & Analytics
- User Permissions
- Billing Integration
